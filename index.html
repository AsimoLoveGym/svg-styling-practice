1. Originally filled. Easiest to work with. Simply apply fill/stroke as desired in CSS to the instance

  • No inline fill; CSS Class works
  • But is from filled instance. Also, we can still apply a stroke

SVG Only

SVG + CSS class

.filled-instance {
  stroke: #cc8ac1;
  stroke-width: 5px;
  fill: lighten(#cc8ac1, 20%);
}

2. Originally stroked with no fill using [stroke] attribute in CSS to “reach-in” to the SVG

  • stroked instance is no longer styleable from instance
  • Inline Fill none + Stroke; CSS class styling not working;
  • But symbol#xxx [stroke] works

SVG Only

path 
  fill="none" 
  stroke="#000000"

SVG + CSS class

.stroked-instance {
  stroke: green; 
  fill: red;
}

SVG + CSS class (symbol#svgname[stroke])

symbol#completed-copy-stroked-2 [stroke] {
  stroke: #dd6435;
  stroke-width: 5px;
}

3. Originally filled, but fill turned off

  • You can always achieve an outlined effect by simply "turning off" the fill
  • No inline fill (default black); CSS Class fill transparent works

SVG Only

SVG + CSS class

.filled-instance-off {
  stroke: #d08aaf;
  stroke-width: 5px;
  fill: transparent;
}

4. Manually adding classes to the source SVG allows us to style to those in CSS

  • You can also just add classes within the source SVG and apply CSS to those directly.
  • SVG Inline classes

SVG Only

  path fill="none" stroke="#000000"
  path class="ibestrokin" fill="none"
  path class="ibestrokin" fill="none"
  polygon class="istrokeittotheeast" fill="none" stroke="#000000"
                    
.ibestrokin {
    stroke: magenta;
    stroke-width: 5px;
  }
                    
.istrokeittotheeast {
    stroke: green;
    stroke-width: 7px;
}

5. Hand Filled–Note that the part of the filled area that's been defined with a fill attribute (red), will NOT get overriden when we apply CSS to the SVG cloned instance. The parts that are working are cyan.

  • Inline fill red > css styling cyan; but other place works
  • Although we can override fill/stroke attributes from CSS if we apply that CSS directly to the definition, we cannot do via CSS applied to the instance

SVG Only

SVG + CSS class

.hand-filled-instance {
    fill: cyan; //Not working with parts filled
    stroke: #CC0000;
    stroke-width: 5px;
}
